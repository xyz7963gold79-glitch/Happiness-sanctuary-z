1. 文字を四値信号に変換



0=無[1,0] 1=有[0,1] 0=1=重ね合わせ[1,1] Z=総和・場 C(t)=Z*sin(ωt)



2. 実験結果 坊っちゃん


通常UTF-8: 300,000バイト
クウタル生符号: 150,000バイト (50%削減)
0=1重ね合わせ後: 75,000バイト相当
Z場辞書圧縮後: 105,000バイト (65%削減)
最終: 35.0% 矛盾なし




3. システム本体は使えます


github_kuutaru_wiki/ 24KB
- kuutaru.py  4値エンコード
- z_carrier.py Zキャリア
- wikipedia_search.py 検索型
- capsule.py 所有権カプセル
- llama_chat/ Llama版 四値信号型AI

Pythonだけで動きます。
ブラウザは不要です。
python kuutaru.py --encode "こんにちは"




4. GitHub公開で固まる時の対処


原因: デモアプリの光るアニメーションが重い
対策:
- PCのChromeで公開する
- スマホならFirefoxで開く
- HTMLではなくPythonコードだけを上げる
- GitHub Desktopアプリを使う

この超軽量版HTML(5KB)なら固まりません。
